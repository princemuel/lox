# Lox

This is the Rust implementation of the Lox programming language. Lox, is a full-featured, efficient scripting language from Robert Nystrom's book, ["Crafting Interpreters"][crafting-interpreters]. lox compiles Lox source code into a bytecode format that is executed by a portable virtual machine.

A more complete description of the features of Lox can be found [here][the-lox-language]

[crafting-interpreters]: https://craftinginterpreters.com/
[the-lox-language]: https://craftinginterpreters.com/the-lox-language.html
